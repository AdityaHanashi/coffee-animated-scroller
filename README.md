# ☕ Cinematic Coffee Website

A premium, interactive coffee landing page built with modern frontend technologies and smooth cinematic animations.

---

## 🚀 Features

* 🎬 Cinematic hero section with splash animation
* 🎯 Scroll-based storytelling using GSAP ScrollTrigger
* 🎨 Split background design (chocolate + light theme)
* ☕ Animated coffee mug with rotation and movement
* 💫 Smooth transitions and micro-interactions
* 📱 Responsive modern layout

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
* GSAP (GreenSock Animation Platform)
* ScrollTrigger

---

## ⚡ How It Works

* On page load, the coffee mug appears with a splash animation
* As the user scrolls:

  * The mug moves downward
  * Rotates smoothly
  * Fades out and disappears
* The hero section is pinned to create a cinematic storytelling effect
